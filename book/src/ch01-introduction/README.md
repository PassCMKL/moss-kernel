# Chapter 1: Introduction

An operating system is the most fundamental software that runs on a computer. It is the layer that transforms bare hardware — a CPU, some RAM, a disk — into a useful platform for running applications. Every time you open a browser, save a file, or launch a terminal, you are relying on the operating system to mediate that interaction with the hardware on your behalf.

This chapter introduces the core ideas behind operating systems and explains where Moss fits in the landscape of OS kernels.

## Learning Objectives

By the end of this chapter you should be able to:

- Describe the roles an operating system plays
- Distinguish between monolithic, microkernel, and other kernel architectures
- Explain why Moss is written in Rust and what that buys us
- Navigate the Moss source tree and find the code for a given subsystem
