# Chapter 11: Building and Testing Moss

This chapter walks through setting up a development environment, building the kernel, running it in QEMU, and running the various test suites.

## Learning Objectives

By the end of this chapter you should be able to:

- Set up the required toolchain
- Build the kernel and create a disk image
- Run Moss in QEMU interactively
- Run unit tests, kernel tests, and userspace tests
